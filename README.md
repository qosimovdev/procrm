# CRM Project Management System

A modern CRM system for managing projects, clients, tasks, and team workflows.

---

## Overview

This project is a project management CRM dashboard designed to help teams organize work, track progress, manage budgets, and handle tasks in one place.

---

## Features

- Project creation and management
- Client management
- Task tracking system
- Team members management
- Budget and spending tracking
- Status and priority handling
- Deadline management
- Activity tracking
- Dashboard analytics

---

## Tech Stack

- React
- JavaScript (ES6+)
- Tailwind CSS
- Shadcn/UI
- Sonner (notifications)

---

## Installation

```bash
git clone https://github.com/qosimovdev/procrm
npm install
npm run dev
```
